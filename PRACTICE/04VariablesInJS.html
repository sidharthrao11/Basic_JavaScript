Varibales In JS

VARIABLE :- where we can store the value/data, basically its identify the storage name of values/data.

Rules of declairation :- (a) variable name should'nt be started from digit(number).
(b) white space will be not allowed.
(c) symbols will be also not allowed but we can use only two special symbols : $(dollar sign) , and _ (underscore).
(d)Its case sensitive.
(e) We can declair variable first and can assign value later.


We've three types of variables in JS :-

1.Var :- we can declair and assign variable name and data multiple times.
i.e. :- var myname = "sidharth Rao";
var myname = "sidharth Rao";
document.write(myname);
NOTE :- IT WILL RUN.


2.let :- cant declair multiple times but, can assign values multiple times.
let myname = "sidharth Rao";
let myname = "sidharth Rao";
document.write(myname);
NOTE :- IT WILL SHOW AN ERROR THAT VARIABLE NAME IS ALREADY DECLAIERD.


3.const :- cant declair multiple times also cant assign multiple times.
const myname = "sidharth Rao"; const myname = "sidharth Rao"; document.write(myname); NOTE :- IT WILL THROW ERROR THAT CANT ASSIGN VALUE TWICE NEIGHTER DECLAIR SAME VARIABLE AGAIN..
